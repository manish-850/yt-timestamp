# Install YT Timestamp Locally

## Prerequisites

- Chrome, Edge, or any Chromium-based browser
- A working copy of this extension repository

## Install steps

1. Open your browser and go to `chrome://extensions/`.
2. Enable `Developer mode` in the top-right corner.
3. Click `Load unpacked`.
4. Select the project folder.
5. The extension should appear in your browser toolbar and be available on YouTube.

## Usage

1. Open a YouTube video page.
2. Click the injected note button in the player controls, or press `Ctrl + Q`.
3. Enter a note and save it.
4. Open the extension popup to view, delete, or jump to saved notes.
